Небольшое приложение на React. Берёт задачи из публичного API jsonplaceholder,
показывает первые 15. Можно искать по названию и фильтровать по статусу,
по клику на задаче открывается модалка с деталями задачи и данными пользователя.

## Запуск

Нужен установленный node (у меня 22, но на 18+ тоже должно работать).

```
npm install
npm run dev
```
Далее перейти по ссылке,которую покажет vite в терминале.

## Что использовал

- React + Vite
- MUI для интерфейса
- fetch для запросов

## Что сделано

- загрузка задач из API, первые 15
- поиск по названию
- фильтр по статусу: все / выполненные / невыполненные
- кнопка сброса фильтров и счётчик найденных
- детальная карточка в модалке (id, статус, userId + имя и email пользователя)
- состояния загрузки / пусто / ошибка
